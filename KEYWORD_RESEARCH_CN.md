# SellCopyTools 关键词研究计划

更新时间：2026-05-15

## 结论先说

当前方向可以继续，但不能靠泛词硬打。

```text
不要主攻：product description generator
优先主攻：平台 + 具体文案类型 + generator/template/example
```

原因：

- 泛词搜索量可能更大，但竞争对手强。
- 长尾词更贴近真实用户需求。
- 长尾词更适合新站从 0 开始积累展示和点击。
- 我们现有页面已经围绕 ecommerce copy tools，适合继续深挖 Shopify、Amazon、Etsy、广告文案、邮件文案。

## 第一阶段关键词策略

第一阶段目标不是追求最大搜索量，而是找到新站能进入结果页的机会。

筛选标准：

```text
用户意图明确：用户确实想生成文案或找模板
页面能承接：我们能做出对应工具页或指南页
商业价值明确：用户是卖家、店主、运营、广告投放者
竞争不要过强：搜索结果不是全被大品牌和平台官方占满
可以系列化：一个关键词能扩展成多个相关页面
```

## 第一周优先做什么

第 1 天：

```text
确认 GA4 和 Search Console 正常。
确认 sitemap 已提交。
建立关键词池。
不要急着申请 AdSense。
```

第 2-3 天：

```text
新增 2-3 个 P2 工具页。
优先做：
1. Shopify Meta Description Generator
2. Amazon Product Title Generator
3. Etsy Title Generator
```

第 4-5 天：

```text
继续新增 2 个工具页。
优先做：
1. Ecommerce SEO Title Generator
2. Product Feature Generator
```

第 6-7 天：

```text
新增 1 篇指南页。
建议标题：
How to Write Product Descriptions for Shopify
```

这篇文章不要只是工具页，要写真实结构：

```text
什么是好的 Shopify 产品描述
产品描述结构
示例
常见错误
如何用工具生成初稿
```

## 手动验证关键词的方法

打开：

```text
https://www.google.com
```

搜索每个关键词，看前 10 个结果：

```text
是否都是大品牌？
是否有小工具站？
是否有页面很薄但排名靠前？
是否有 Reddit、论坛、低质量目录页？
是否搜索结果标题都很接近？
```

如果结果里出现小工具站、薄页面、目录页，说明新站有机会。

如果结果全是 Shopify 官方、Amazon 官方、大型 SaaS、大媒体，短期不优先。

## 使用 Google Trends

打开：

```text
https://trends.google.com/
```

操作：

```text
输入 3-5 个关键词对比。
地区先看 United States，也可以看 Worldwide。
时间选择 Past 12 months 和 Past 5 years 各看一次。
```

优先看：

```text
趋势是否长期存在
是否有明显下降
相关查询里有没有更具体的长尾词
```

## 使用 Google Ads Keyword Planner

打开：

```text
https://ads.google.com/
```

操作路径：

```text
Tools
-> Keyword Planner
-> Discover new keywords
```

建议设置：

```text
Location: United States
Language: English
Search network: Google
```

输入种子词：

```text
shopify product description generator
amazon product description generator
etsy product description generator
amazon bullet point generator
abandoned cart email generator
```

导出后重点看：

```text
Avg. monthly searches
Competition
Top of page bid
```

不是搜索量越大越好。新站优先选择：

```text
搜索量有一点
竞争不要太极端
商业意图明确
我们能做出更具体页面
```

## 当前不建议做的方向

暂时不要开新域名。

除非方向完全不同，例如：

```text
图片编辑器
PDF 工具
计算器
小游戏
法律模板
简历模板
```

只要还是 ecommerce copy、产品文案、广告文案、邮件文案，都继续放在：

```text
sellcopytools.com
```

这样可以积累同一个域名的内容厚度和主题相关性。

## 30 天判断标准

30 天内重点看：

```text
Search Console 是否开始出现 impressions
哪些页面先被索引
哪些关键词有展示
Analytics 是否有非自己访问
是否有页面停留或重复访问
```

如果 30 天完全没有 impressions：

```text
减少泛词工具页
增加更具体的长尾页
增加 guide / examples / templates 页面
检查页面是否太薄
检查是否有抓取或索引问题
```

## 90 天判断标准

坚持 60-90 天再判断方向，不要第一周就否定。

继续条件：

```text
有 impressions 增长
部分页面被索引
有少量自然访问
能找到更多长尾词
内容生产能持续
```

需要调整条件：

```text
90 天几乎没有展示
所有关键词都被强站压制
工具体验没有明显差异
页面无法扩展出内容深度
```
